> 技术栈: Vue 3 + Element Plus + Vite + Pinia + Vue Router

## 关键交互说明

1. **脚本 → 分镜联动**: 脚本保存后，分镜Tab自动提示"脚本已更新，是否重新生成分镜？"
2. **分镜卡片点击**: 展开右侧抽屉，显示/编辑镜头详情(image_prompt, video_prompt等)
3. **生成任务轮询**: 点击生成后，前端每3秒轮询任务状态，更新进度条
4. **快捷工作流**: 仪表盘/项目列表支持"一键生成"——从脚本到最终视频的完整流水线
